## 🛠️ I'm currently working on

**ShipShip ⛴️** — an open-source dynamic project creator built with Next.js and Payload CMS. Leveraging Next.js's clean structure and Payload CMS's simplicity, the goal is to create a boilerplate where, with just a few selections (e.g., authentication, database, emailing, blogs, etc.), you can instantly generate a project template that's already close to production-ready.

No more starting from scratch — just pick what you need and set sail 🚀.

## 🚀 Launched Projects

### [GitDynamic](https://gitdynamic.io)
A dynamic GitHub profile and portfolio enhancement platform. GitDynamic lets developers create richer, data-driven GitHub profiles by surfacing real-time stats, project highlights, and custom content — making your GitHub presence genuinely stand out.

### [PsychMem](https://github.com/muratg98/psychmem)
A psychology-grounded selective memory system for AI agents. Inspired by how human memory actually works, PsychMem brings short-term/long-term memory consolidation, decay curves, and learned importance weights to AI tools like Claude Code and OpenCode — giving agents smarter, more human-like recall.
